What is Mare (Math animation realtime engine)?

Mare is a realtime simulation engine fully written in Javascript that requires some basic scripting to make beautifull and meaninfull animation that show the beauty of our life's rules (math and physics) and can be also used to verify geometric proofs and formulas. have fun ! here are some examples :
